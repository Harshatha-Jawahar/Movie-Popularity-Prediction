# Movie Popularity Prediction using Scikit-Learn

## Project Overview
This is a simple machine learning mini project developed using Python and Scikit-Learn.
The objective of this project is to predict whether a movie will be classified as
Hit or Average based on selected features.

## Features Used
- Budget
- Duration
- IMDb Rating

## Algorithm Used
- K-Nearest Neighbors (KNN)

## Tools and Technologies
- Python
- Jupyter Notebook
- Scikit-Learn
- Pandas

## Dataset
The dataset is manually created within the Jupyter Notebook to maintain originality.
It contains basic movie attributes and their corresponding popularity labels.

## Output
The model predicts movie popularity as either Hit or Average.
The accuracy of the model is also calculated.

## Conclusion
This project demonstrates a basic classification model using Scikit-Learn and
helps in understanding the fundamentals of machine learning.
